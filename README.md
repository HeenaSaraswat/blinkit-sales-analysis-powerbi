# blinkit-sales-analysis-powerbi
Power BI project that transforms Blinkit sales data into actionable business insights through data modeling, DAX calculations, and dynamic visualizations.
## 🎯 Objectives
- Analyze overall sales performance.
- Identify top-performing products and categories.
- Compare sales across different outlet types and locations.
- Track key business KPIs.
- Build an interactive dashboard for business decision-making.

---
## 🛠️ Tools & Technologies
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Microsoft Excel / CSV

---
## 📂 Dataset
The project uses the Blinkit Grocery Sales dataset containing multiple related tables.

### Tables Used
- BlinkIT Grocery Data
- Items
- Items Content
- Outlet Info
- Outlet Location
- Cities
---
## 🗂️ Data Model
The project follows a **Star Schema** where:
- **Fact Table:** BlinkIT Grocery Data
- **Dimension Tables:**
  - Items
  - Items Content
  - Outlet Info
  - Outlet Location
  - Cities
